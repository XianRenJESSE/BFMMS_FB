安全操作定理推理：

前提：所有操作数运算时均不允许溢出回绕，一旦溢出即视为编码错误，立即终止进程

一. 当双方都是代理指针时：
定义：
全集1  为 源操作代理指针代理区域    记作 U1 = [UB1, UE1)  |U1| = S1
全集2  为 目标操作代理指针代理区域  记作 U2 = [UB2, UE2)  |U2| = S2
集合O1 为 源操作区间              记作 O1 = [B1, E1)    |O1| = E1 - B1 = S3
集合O2 为 目标操作区间            记作 O2 = [B2, E2)    |O2| = E2 - B2 = S4
空集   为 E
空指针 为 N

命题：
1. 不会操作空地址 <==> &U1 != N, &U2 != N

2. 有可操作的数据 <==> U1 != E, U2 != E <==> S1 > 0, S2 > 0

3. 不存在部分重叠管理的空间
<==>  U1 与 U2 的交集等于空集E 或者 U1 包含 U2 或者 U2 包含 U1 或者 U1 等于 U2
<==> 以上条件过多，取以上条件的否定形式：
即：有一个区间的一个端点在另一个区间内，同时另一个端点在另一个区间外 的 否定
<==> ！(U1 与 U2 的交集不等于空集E 并且 U1 是 U2 的真子集 并且 U2 是 U1 的真子集)
<==>
(
    UB1 < UB2                     //UB1 < UB2 如果取等就内含了
    &&
    (UB2 < UE1 && UE1 <= UE2)     //UB2 < UE1 开与闭相邻不能取等
)
||
(
    (UB2 <= UB1 && UB1 < UE2)     //UB1 < UE2 开与闭相邻不能取等
    &&
    UE2 < UE1                     //UE2 < UE1 如果取等就内含了
)
如 [0,10) [7,17) 这就是部分重叠，无法安全操作，不合法的代理指针

4. 操作的区间有效：
<==> U1 包含 O1 并且 U2 包含 O2 并且 O1 != E 并且 O2 != E
<==> (UB1 <= B1 && E1 <= UE1) && (UB2 <= B2 && E2 <= UE2) && (S3 != 0 && S4 != 0)

5. 没有原地操作数据
<==> B1 != B2 && E1 != E2

6. 不存在重叠操作的范围
<==> O1 与 O2 的交集等于空集E
<==> E1 <= B2 || E2 <= B1

二. 当有一方是void*指针时：
定义：
全集1  为 目标操作代理指针代理区域  记作 U = [UB, UE)  |U| = S1
集合V  为 void*指针操作区域        记作 V = [VB, VE)  |V| = S2
集合O  为 源操作区间              记作 O = [B, E)    |O| = E - B = S3
空集   为 E
空指针 为 N

命题：
1. 不会操作空地址 <==> &U != N, &V != N

2. 有可操作的数据 <==> U != E, V != E <==> S1 > 0, S2 > 0

3. 代理指针管理的空间 与 void*指针操作的空间 不 部分重叠
   且 void*操作区间 不等于 代理指针管理的空间 //原地移动数据没有意义
<==> U 与 V 的交集等于空集E 或者 U 包含 V
<==> (VE <= UB || UE <= VB) || (UB <= VB && VE < UE) || (UB < VB && VB <= UE)
<==> 以上条件过多，取以上条件的否定形式：
     即：有一个区间的一个端点在另一个区间内，同时另一个端点在另一个区间外 的 否定
<==>
(
    VB < UB
    &&
    (UB < VE && VE <= UE)
)
||
(
    (UB <= VB && VB < UE)
    &&
    UE < VE
)
//细节同上"一.3"

4. 操作的大小有效 <==> (S1 >= S2) && (S3 != 0)

5. 操作的区间有效 <==> U 包含 O <==> UB <= B && E <= UE

6. 不存在重叠操作的空间
<==> O 与 V 的交集等于空集E
<==> E1 <= B2 || E2 <= B1
